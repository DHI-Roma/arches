### Breaking changes

- The minimum supported version of Python is now 3.12.

### Deprecations

### Upgrading Arches

1. You must be upgraded to at least version 8.1.0 before proceeding. 

2. Upgrade to Arches 8.2.0
    ```
    pip install --upgrade arches==8.2.0
    ```

### Upgrading an Arches project

1. In pyproject.toml, make these updates:
    - Update `requires-python` to `">=3.12"`
    - Remove `"Programming Language :: Python :: 3.11"`
    - Remove `"Framework :: Django :: 5.2"`
    - Add `"Framework :: Django :: 6.0"`

1. Apply automatic updates:

    ```
    python manage.py updateproject
    ```

1. In package.json:

    Remove "arches-dev-dependencies" from "devDependencies" and run:


        ```bash
        rm -r node_modules; rm package-lock.json
        npm install
        ```

1. Apply automatic updates:

    ```bash
    python manage.py updateproject
    ```
