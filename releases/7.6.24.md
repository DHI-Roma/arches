## Arches 7.6.24 Release Notes

### Bug Fixes

-  Allows JSON-LD reader to import xsd:dateTime into a graph node defined as an rdfs:Literal[#12716](https://github.com/archesproject/arches/pull/12716)
-  Add missing datatype specific "to_rdf", "from_rdf", and "is_a_literal_in_rdf" methods to the EDTF datatype[#12717](https://github.com/archesproject/arches/pull/12717)

### Dependency changes:

```
Python:
    Upgraded:
        None
JavaScript:
    Upgraded:
        None
```

1. Upgrade to version 7.6.0 before proceeding by following the upgrade process in the [Version 7.6.0 release notes](https://github.com/archesproject/arches/blob/dev/7.6.x/releases/7.6.0.md)

1. Upgrade to Arches 7.6.24

    ```
    pip install --upgrade arches==7.6.24
    ```

1. If you are running Arches on Apache, restart your server:
    ```
    sudo service apache2 reload
    ```
