## Arches 7.6.24 Release Notes

### Bug Fixes

-  Fixes JSON-LD writer incorrectly exporting date nodes as xsd:dateTime instead of what is actually modeled in the graph[#12716](https://github.com/archesproject/arches/pull/12716)

### Dependency changes:

```
Python:
    Upgraded:
        None
JavaScript:
    Upgraded:
        None
```

1. Upgrade to version 7.6.0 before proceeding by following the upgrade process in the [Version 7.6.0 release notes](https://github.com/archesproject/arches/blob/dev/7.6.x/releases/7.6.0.md)

1. Upgrade to Arches 7.6.22

    ```
    pip install --upgrade arches==7.6.24
    ```

1. If you are running Arches on Apache, restart your server:
    ```
    sudo service apache2 reload
    ```
