### Upgrading Arches

1. Upgrade to version 8.0.0 before proceeding by following the upgrade process in the [Version 8.0.0 release notes](https://github.com/archesproject/arches/blob/dev/8.0.x/releases/8.0.0.md)

1. Upgrade to Arches 8.0.9
    ```
    pip install --upgrade arches==8.0.9
    ```

1. Run the following SQL patch files

    You can either run it from the command line:
    ```
    psql -U <username> -d <database_name> -f "{path to the root of the arches install}/arches/app/db/patches/0001_v8.0_tile_cardinality_check.sql"
    ```
    
    .. or you can copy it from [here](https://github.com/archesproject/arches/blob/dev/8.0.x/arches/db/patches/0001_v8.0_tile_cardinality_check.sql) and run it in your database client.


1. Rebuild your frontend bundle:

    - If you plan on doing active development, and need to see frontend changes automatically, run:

        ```
        npm start
        ```

    - If you do not plan to do active development, run:

        ```
        npm run build_development
        ```

    - if you are running in production, collect your static files
        ```
        python manage.py collectstatic
        ```

1. If you are running Arches on Apache, restart your server:
    ```
    sudo service apache2 reload
    ```
