### Upgrading Arches

1. Upgrade to at least version 8.1.0 before proceeding by following the upgrade process in the [Version 8.1.0 release notes](https://github.com/archesproject/arches/blob/dev/8.1.x/releases/8.1.0.md)

1. Upgrade to Arches 8.1.3

    ```bash
    pip install --upgrade arches==8.1.3
    ```

### Upgrading an Arches project

1. Rebuild your frontend bundle:

    - If you plan on doing active development, and need to see frontend changes automatically, run:

        ```bash
        npm start
        ```

    - If you do not plan to do active development, run:

        ```bash
        npm run build_development
        ```

    - if you are running in production, collect your static files:

        ```bash
        python manage.py collectstatic
        ```

1. If you are running Arches on Apache, restart your server:

    ```bash
    sudo service apache2 reload
    ```
